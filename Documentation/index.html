A) HTML Structure - top

This is a one page landing page. All of the information within the main content area is nested within a div with an id of "primaryContent". The general template structure is the same throughout the template. Here is the general structure.

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

B) CSS Files and Structure - top

The first CSS file, default.css contains all the specific styles for this page. It also displays a table of contents at the top like this:

	
/*

	01. Generals Settings
	02. Header Section Settings

	06. Media Queries for Mobile and Tablets Settings

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

There are additional css files, each specific to a different plugin or framework used for this plugin.

C) JavaScript - top

This theme imports four Javascript files.

  1. jQuery
  2. Bootstrap
  3. jQuery easing
  4. jQuery nicescroll
  5. Owl Carousel
  6. My custom scripts
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. script.js contains the custom scripts needed for this page: Fading effects for the top part, email verification for subscription etc.

D) Subscription Form - top

For using it, you'll have to add your email address in contact.php and book.php file, on line 3. 

$EmailFrom = "no-reply@example.com"; // replace with your domain address
$EmailTo = array ( "mail@example.com"); // your email address, where to receive new subscriptions
